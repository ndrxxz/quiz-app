# 🧠Quiz App
This quiz app is a simple interactive web application created with HTML, CSS, and JavaScript. The app displays questions sequentially, and users can navigate through them by clicking the ‘Next’ button. A timer is also implemented to add challenge and manage the time for answering each question.

## ✨Technologies
- HTML
- CSS
- JavaScript

## 🚀Features
- Multiple choice questions
- Next button for navigating questions
- Countdown timer for each question
- Shows the correct answer when time runs out
- Simple and interactive interface

## 💡What I Learned
In this project, I learned how to build a simple quiz application using HTML, CSS, and JavaScript. I used HTML to create the structure of the quiz, such as the questions, answer choices, and buttons. I used CSS to design the layout and make the quiz visually appealing and easy to use. With JavaScript, I learned how to add interactivity by checking the user's answers, updating the score, and moving to the next question. I also learned how to use functions, variables, and conditional statements to control how the quiz works. This project helped me improve my problem-solving and web development skills.

## 🚦Running the Project
- Clone the repository
- Open the project using a live server (or any local server)
- This app will automatically open in your browser at: `http://127.0.0.1:5500/index.html`
